How it works
01

Pick a folder

Choose any folder to organize — your Desktop, Downloads, Documents. Hvtch scans it and shows you what's inside.

02

Create a rule

Type what you want in plain English — "Move old screenshots to a Screenshots folder" — or build a rule manually. Either way, it runs locally.

03

Review and execute

Hvtch shows you exactly what will happen before it does anything. Approve the matches, uncheck what you want to keep. You're always in control.

04

Let it run

Set rules to run on a schedule — hourly, daily, or weekly. Hvtch watches from the menu bar. Every move is undoable.

Files stay on your Mac

Every scan, every rule, every file move runs locally. No cloud processing. Works fully offline.

AI never sees your files

The AI reads your typed descriptions only. Never filenames, paths, contents, or any metadata from your Mac.

Everything is undoable

Every file move is tracked. Restore one file or undo an entire batch. Nothing is ever deleted.

Describe what you want. Hvtch builds the rule.

Type a sentence. Get a rule that runs locally, forever, with zero network dependency. Uses your own OpenAI or Anthropic key.

when name contains "Screenshot"
  and older than 30 days

Sort by type, age, size, or name

Move old screenshots to a folder. Archive downloads older than 30 days. Organize PDFs, images, documents — whatever's piling up.

Teach Me Once

When Hvtch finds files with no rule, it asks what to do — once. Your answer becomes a permanent rule.

Scheduled scans

Set rules to run hourly, daily, or weekly. Hvtch watches from the menu bar and notifies you when files move.

Organize folders too

Sort folders by name, age, or project type — Xcode, Node.js, Python, and more. Not just files.

Metadata only

Reads filenames, sizes, and dates — the same info you see in Finder. Never opens or analyzes file contents.
